# 📄 Office to PDF Tool

A simple, one-click Windows application that converts Microsoft Office documents to PDF format. Supports **Word**, **Excel**, and **PowerPoint** files. Built for family and friends who need a hassle-free converter without installing Python or any dependencies.

---

## ✨ Features

- **Multi-Format Support** – Word (.docx, .doc, .rtf, .odt), Excel (.xlsx, .xls, .csv), PowerPoint (.pptx, .ppt)
- **Simple GUI** – Clean, user-friendly interface
- **One-Click Conversion** – Select a file, click convert, done
- **No Dependencies** – Standalone `.exe` file, nothing to install
- **Safe Conversions** – Works even when the source file is open in Office
- **Auto-Detection** – Automatically recognizes file type and uses the right Office app
- **Overwrite Protection** – Asks before replacing existing PDFs
- **Progress Feedback** – Loading animation during conversion

---

## 📥 Download & Use

### Option 1: Download the .exe (Recommended for Everyone)

1. **[Click here to download FamilyPDFTool.exe](../../raw/main/FamilyPDFTool.exe)**
2. Save the file anywhere (Desktop is fine)
3. Double-click to run
4. Select your Office document and click **Convert to PDF**

✅ That's it! No Python, no installers, no command line.

### Option 2: Download the Whole Project

1. Click the green **Code** button above
2. Select **Download ZIP**
3. Extract and run `FamilyPDFTool.exe`

## 📋 Supported Formats

| Category | Formats |
|----------|---------|
| **Word** | `.docx` `.doc` `.docm` `.dotx` `.dotm` `.rtf` `.odt` |
| **Excel** | `.xlsx` `.xls` `.xlsm` `.xltx` `.xltm` `.csv` |
| **PowerPoint** | `.pptx` `.ppt` `.pptm` `.potx` `.potm` |

> 💡 The app automatically detects your file type and uses the correct Office program.

## ⚙️ Requirements

| Requirement | Needed? | Notes |
|------------|:-------:|-------|
| **Microsoft Word** | ⚠️ | Required for Word documents |
| **Microsoft Excel** | ⚠️ | Required for Excel spreadsheets |
| **Microsoft PowerPoint** | ⚠️ | Required for PowerPoint presentations |
| Microsoft Office | ✅ | At least the apps you want to convert from |
| Python | ❌ | Not needed |
| Internet | ❌ | Not needed |
| Admin rights | ❌ | Not needed |

> ⚠️ **Important:** This tool uses Microsoft Office behind the scenes. You only need the Office apps installed for the file types you want to convert. For example, if you only convert Word documents, you only need Word.
